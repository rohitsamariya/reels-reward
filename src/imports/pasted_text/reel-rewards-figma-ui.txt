Act as a senior SaaS product designer and create a complete interactive Figma UI for a web application called "ReelRewards".

This platform allows users to earn money by watching Instagram reels.

IMPORTANT REQUIREMENTS

• The design must be mobile-first (primary focus on smartphone users)
• Desktop layout should also exist but as a secondary layout
• All screens must look realistic and functional with dummy data
• Use modern card-based mobile UI similar to Instagram style apps
• Include interactive prototype navigation between screens

---

THEME & VISUAL STYLE

Design theme inspired by Instagram colors.

Primary gradient:
Red (#FF3B5C)
Orange (#FF7A18)
Pink (#E1306C)

Use warm red/orange shades and gradients similar to Instagram branding.

Typography:
Inter / SF Pro style

Mobile design rules:

• Large readable fonts
• Card-based layout
• Rounded UI elements
• Bottom navigation bar for mobile
• Touch-friendly buttons
